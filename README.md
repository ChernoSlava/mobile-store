# My Electron App

A sleek desktop application built with **Electron**, **JavaScript**, **HTML5**, and **CSS3**. This project demonstrates a functional desktop environment for managing data and tasks.

## 🚀 Features
* **Native Desktop Window:** High-performance container using Chromium.
* **Local Data Storage:** Interaction with local JSON files (e.g., `phones.json`).
* **Modular Codebase:** Separation of concerns between the Main process (`main.js`) and Renderer process (`renderer.js`).
* **Cross-Platform Ready:** Configured for macOS (arm64/Intel) and ready for Windows/Linux builds.

## 🛠 Prerequisites
Before you begin, ensure you have the following installed:
* [Node.js](https://nodejs.org/) (LTS version recommended)
* npm (comes with Node.js)
